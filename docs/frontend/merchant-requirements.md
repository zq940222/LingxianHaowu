# 商家端需求分析

## 一、功能模块

- 商品状态筛选（上架/下架）
- 分类筛选
- 批量操作（上架/下架/删除）
- 搜索商品

#### 1.2 商品添加/编辑
- 商品基本信息（名称、分类、价格、库存）
- 商品图片上传
- 商品规格设置
- 商品描述
- 设置推荐商品
- 设置热卖商品
- 设置拼团商品及拼团价
- 拼团人数设置

### 2. 订单管理模块
#### 2.1 订单列表
- 全部订单
- 待处理订单
- 待发货订单

- 订单搜索

#### 2.2 订单详情
- 订单信息展示
- 收货地址
- 配送方式/时间
- 订单商品
- 支付状态
- 处理订单
- 发货
- 配送管理

#### 2.3 订单操作
- 接单
- 拒单（备注原因）
- 发货
- 完成订单
- 取消订单

### 3. 配送管理模块
#### 3.1 配送小区管理
- 添加配送小区
- 编辑配送小区
- 删除配送小区
- 设置配送费
- 设置起送金额
- 设置配送时间

#### 3.2 自提点管理
- 添加自提点
- 编辑自提点
- 删除自提点
- 设置自提时间

#### 3.3 配送安排
- 查看待配送订单
- 按小区/自提点分组
- 配送路线规划
- 配送员分配

### 4. 用户管理模块
#### 4.1 用户列表
- 用户信息展示
- 用户搜索
- 用户筛选（新用户/活跃用户）

#### 4.2 用户详情
- 用户基本信息
- 订单记录
- 积分记录
- 积分调整（充值/扣除）

### 5. 活动管理模块
#### 5.1 活动弹窗配置
- 创建活动弹窗
- 编辑弹窗内容
- 设置弹窗展示时间

- 弹窗预览

#### 5.2 优惠券管理
- 创建优惠券
- 设置优惠券规则（满减/折扣/无门槛）
- 设置有效期

#### 5.3 促销活动
- 创建促销活动
- 设置活动商品
- 设置活动时间

### 6. 数据统计模块
#### 6.1 销售统计
- 今日销售额
- 今日订单数
- 今日客单价

#### 6.2 商品统计
- 热销商品排行

- 商品库存预警

#### 6.3 用户统计
- 新增用户数
- 活跃用户数

### 7. 库存管理模块
- 库存预警设置
- 库存盘点

### 8. 个人中心模块
- 商家信息管理
- 修改密码
- 消息通知
- 帮助中心

## 二、业务流程

### 商品上架流程
```
添加商品信息 → 上传图片 → 设置价格/库存 → 设置商品标签
→ 设置拼团（可选） → 提交审核 → 审核通过 → 上架销售
```

### 订单处理流程
```
接收新订单 → 确认接单 → 备货 → （自提：通知用户取货 / 配送：安排配送）
→ 完成配送 → 用户确认收货 → 订单完成 → 计算积分
```

### 活动配置流程
```
创建活动 → 上传弹窗图片 → 设置活动规则 → 设置展示时间
→ 预览效果 → 发布活动 → 监控数据
```

## 三、商家角色权限

| 角色 | 权限 |
|------|------|
| 超级管理员 | 所有功能权限 |
| 运营人员 | 商品管理、订单处理、数据统计 |
| 配送人员 | 订单配送、配送路线规划 |

## 四、非功能需求

### 性能要求
- 订单列表加载 < 2秒
- 商品列表加载 < 1.5秒
- 数据统计查询 < 3秒

### 安全要求
- 商家身份验证
- 操作日志记录
- 敏感信息加密

### 数据准确性要求
- 订单状态实时更新
- 库存数据准确
- 财务数据可追溯
