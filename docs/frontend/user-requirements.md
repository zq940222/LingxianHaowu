# 用户端需求分析

## 一、功能模块

### 1. 首页模块
- 推荐商品展示（轮播图/网格）
- 热卖商品展示
- 拼团商品展示
- 活动弹窗（优惠券、促销活动）
- 快捷入口（分类、搜索、购物车）

### 2. 商品模块
#### 2.1 商品列表
- 分类筛选（蔬菜、水果、肉类等）
- 热卖商品筛选
- 拼团商品筛选
- 价格排序
- 分页加载

#### 2.2 商品详情
- 商品图片展示
- 价格信息（原价/拼团价）
- 商品规格选择
- 库存信息
- 商品描述
- 添加购物车
- 立即购买

### 3. 拼团模块
- 发起拼团
- 参与拼团
- 拼团进度展示
- 拼团好友列表
- 拼团倒计时

### 4. 购物车模块
- 购物车商品列表
- 修改数量
- 删除商品
- 选中/取消选中
- 合计金额计算
- 批量结算

### 5. 订单模块
#### 5.1 下单流程
- 收货地址选择
- 配送方式选择（自提/定时配送）
- 配送时间选择
- 优惠券选择
- 订单确认
- 微信支付

#### 5.2 订单管理
- 订单列表（全部/待付款/待发货/待收货/已完成）
- 订单详情
- 订单状态跟踪
- 取消订单
- 确认收货
- 申请退款

### 6. 配送模块
#### 6.1 配送小区选择
- 可配送小区列表
- 当前配送小区展示
- 切换配送小区

#### 6.2 配送方式
- 自提点选择
- 定时配送时间选择
- 配送费计算

### 7. 积分模块
#### 7.1 签到功能
- 每日签到
- 连续签到奖励
- 签到记录

#### 7.2 积分获取
- 签到获得积分
- 完成订单获得积分
- 积分规则说明

#### 7.3 积分使用
- 积分明细查询
- 积分兑换（优惠券/商品）

### 8. 个人中心模块
- 用户信息展示
- 订单管理入口
- 地址管理
- 积分中心
- 优惠券中心
- 消息通知
- 客服联系

### 9. 活动模块
- 活动弹窗展示
- 优惠券领取
- 促销活动查看
- 活动记录

## 二、业务流程

### 用户下单流程
```
浏览商品 → 添加购物车 → 选择配送方式 → 选择配送时间
→ 选择优惠券 → 确认订单 → 微信支付 → 生成订单
→ 等待发货 → 收货/自提 → 订单完成 → 获得积分
```

### 拼团流程
```
发起拼团 → 分享给好友 → 好友参与拼团 → 达到拼团人数
→ 拼团成功 → 享受拼团价 → 下单支付
（未达到人数 → 拼团失败 → 自动退款）
```

### 积分获取流程
```
每日签到 → 获得签到积分
完成订单 → 根据订单金额计算积分 → 积分到账
```

## 三、用户角色权限

| 角色 | 权限 |
|------|------|
| 普通用户 | 浏览商品、下单、参与拼团、签到、使用积分 |
| 新用户 | 首次下单、领取新用户优惠券 |

## 四、非功能需求

### 性能要求
- 首页加载时间 < 2秒
- 商品列表加载 < 1.5秒
- 支付响应时间 < 3秒

### 安全要求
- 用户信息加密存储
- 支付接口签名验证
- 登录状态校验

### 兼容性要求
- 微信小程序基础库 2.0+
- iOS/Android双端兼容
