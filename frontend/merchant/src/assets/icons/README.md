# TabBar 图标说明

请在此目录添加以下图标文件（尺寸建议 81x81 像素）：

- `home.png` - 工作台图标（未选中）
- `home-active.png` - 工作台图标（选中）
- `order.png` - 订单图标（未选中）
- `order-active.png` - 订单图标（选中）
- `product.png` - 商品图标（未选中）
- `product-active.png` - 商品图标（选中）
- `my.png` - 我的图标（未选中）
- `my-active.png` - 我的图标（选中）

可以使用以下图标库获取图标：
- 阿里巴巴矢量图标库 (https://www.iconfont.cn/)
- Flaticon (https://www.flaticon.com/)
