# Tabbar Icons

请将以下图标文件放置在此目录:

## 需要的图标文件:
- `home.png` - 首页图标 (灰色, 81x81px)
- `home-active.png` - 首页图标 (绿色选中状态, 81x81px)
- `category.png` - 分类图标 (灰色, 81x81px)
- `category-active.png` - 分类图标 (绿色选中状态, 81x81px)
- `cart.png` - 购物车图标 (灰色, 81x81px)
- `cart-active.png` - 购物车图标 (绿色选中状态, 81x81px)
- `my.png` - 我的图标 (灰色, 81x81px)
- `my-active.png` - 我的图标 (绿色选中状态, 81x81px)

## 图标要求:
- 尺寸: 81x81 像素
- 格式: PNG (支持透明背景)
- 普通状态颜色: #999999
- 选中状态颜色: #07c160 (主题绿色)

## 推荐图标来源:
- [Iconfont](https://www.iconfont.cn/)
- [IconPark](https://iconpark.bytedance.com/)
- [Remix Icon](https://remixicon.com/)
